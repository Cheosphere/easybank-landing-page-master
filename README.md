# Landing Page de easybank

El desafío es desarrollar esta landing page y lograr que se parezca lo más posible al diseño.

- HTML
- CSS
- JS Vanilla (with animejs library)

# Guía de Estilo

Los diseños se crearon con los siguientes anchos:

- Mobile: 375px
- Desktop: 1440px
